# Context matters

A code for a research paper submitted to ACL 2025. This repository contains scripts that were used for training and result evaluation. The data itself is not released. 

The data for the analysis was reused from [shared task SMM4H-2024](https://aclanthology.org/2024.smm4h-1.40/). The data is not published and is based on [SMM4H-2017 dataset](https://data.mendeley.com/datasets/rxwfb3tysd/2). To reproduce the experiment results please contact the SMM4H-2024 organizers for getting the access to the original data, then please contact the author of this repository to provide you the GPT-generated contexts.
